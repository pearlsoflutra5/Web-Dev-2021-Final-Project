What is HTML?

HTML stands for HyperText Markup Language. It is a computer language that is used to build websites. HTML creates the structure for the website and the many pages built within it. HTML is the basic blueprint for websites.

For example, HTML is responsible for the existence of:

  • links
  • text
  • buttons
  • subpages
  • images

Essentially, HTML is the tool to insert all of the functioning pieces in a website.

What is CSS?

CSS stands for Cascading Style Sheets. It is a computer language that is used to add design to websites. CSS creates the look for the website and defines how pages respond to different screen types.

For example, CSS is able to:

  • customize colors
  • customize spacing
  • streamline links
  • position elements

CSS is created to be the visual complement for HTML, they are used together to create functional and aesthetically pleasing websites.

Why are these things important?

Webpages are an integral part of our lives. Knowing how web pages work can be helpful in creating new design ideas for the future.

When a person uses HTML and CSS together to create a website is web development. There are several sites online that can help beginners start with templates or drag-and-drop features. In this case, you are not the one creating the website rules you are using predefined ones, however, the principles behind web development are the same.

How can this website help you?

If you are interested in creating websites of your own, this website can act as a starting place to learn what is possible in basic websites. We will run through the basics of HTML, how to implement images and links, and some basic CSS rules.
