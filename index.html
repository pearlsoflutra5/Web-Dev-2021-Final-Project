What is HTML?

HTML stands for HyperText Markup Language. It is a computer language that is used to build websites. HTML creates the structure for the website and the many pages built within it.

What is CSS?

CSS stands for Cascading Style Sheets. It is a computer language that is used to add design to websites. CSS creates the look for the website and defines how pages respond to different screen types.
